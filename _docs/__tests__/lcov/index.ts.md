## Overview

This TypeScript helper runs the project's test suite to produce an LCOV coverage file, converts that LCOV into a Codecov-compatible JSON, and removes temporary coverage artifacts. It declares lcovFile = "__tests__/lcov/lcov.info" and constructs a test command that runs tests with an lcov reporter.

The script executes three sequential async jobs: (1) run the test command via exec and treat non-zero exits as failures; (2) if the lcov file exists, read it, call lcovToCodecov(lcov), ensure __tests__/coverage exists, remove any prior codecov.json, and write the converted JSON; (3) delete the lcov file. Jobs are awaited in order to guarantee serial execution.

## Architecture Notes

- Sequential job pipeline: an array of async anonymous jobs is executed in order by awaiting each job() in a for..of loop, ensuring steps run serially (tests → convert → cleanup).
- File-system based artifact exchange: the script relies on the presence and location of __tests__/lcov/lcov.info and writes output to __tests__/coverage/codecov.json, using fs.existsSync checks to gate work.
- Minimal process supervision: exec is used (not spawn) and stdout/stderr are piped directly; the script treats any non-zero exit code as a failure by rejecting the Promise.

## Usage Examples

### Generate Codecov JSON after running tests

Run this TypeScript file (via node/tsx) from the repository root. The script executes the test runner command defined in testScript (npx tsx with lcov reporter) to produce __tests__/lcov/lcov.info, converts that file to a Codecov JSON using the local ./lcov.js converter, writes the result to __tests__/coverage/codecov.json, and removes the intermediate lcov.info file.

## Maintenance Notes

- The test command is hard-coded to use npx tsx and test reporter flags; ensure tsx is available in the environment or update testScript accordingly.
- Path assumptions are fixed (__tests__/lcov/lcov.info and __tests__/coverage/codecov.json). If test outputs are configured elsewhere, update lcovFile and codecovPath constants.
- The lcov-to-codecov conversion delegates to ./lcov.js; changes to that module's API (export name or signature) require updating the import or call site.
