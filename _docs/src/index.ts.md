## Overview

This module provides a single exported async function build(options?: SuSeeConfig) that orchestrates a Susee build. It first attempts to resolve build configuration by using the provided options parameter; if options is not provided it awaits finalSuseeConfig() to load a root configuration. If neither is available the function logs a colored error message and exits the process with code 1.

Once configuration is resolved, build either converts the passed options using generateBuildOptions(options) or uses the loaded BuildOptions directly, instantiates new Compiler(buildOptions) and awaits compiler.compile(). The function also wraps the compile step in console.time / console.timeEnd using colored labels from the @suseejs/color package. The file also re-exports the SuSeeConfig type.

## Architecture Notes

- Configuration resolution follows a strict priority: explicit function argument (options) → root config via finalSuseeConfig(); if neither exists the process is terminated with process.exit(1).
- The build function performs dependency injection of configuration into Compiler by constructing new Compiler(buildOptions) and awaiting its compile() method, keeping this module as a thin orchestration layer.
- Runtime console timing and colored messages are used for simple visibility; there is no retry or error propagation beyond process.exit for missing configuration.

## Usage Examples

### Main use case

Call build() with an in-memory SuSeeConfig to run a programmatic build, or call build() with no arguments in a CLI context so the module attempts to load a root config via finalSuseeConfig(); on success the Compiler will be constructed and compile() executed, with console timing and colored logs visible in stdout/stderr.

## Maintenance Notes

- Missing configuration results in process.exit(1) — callers that embed this function should be aware it will terminate the Node process in that case.
- finalSuseeConfig() is awaited and may return undefined/null; the code treats absence as a fatal error. Ensure finalSuseeConfig()'s contract is respected by its implementation.
- The file casts an empty object to BuildOptions (let buildOptions = {} as BuildOptions) before assignment; ensure generateBuildOptions and finalSuseeConfig produce complete BuildOptions to avoid runtime issues in Compiler.
